الوحدة الأولى: هياكل البيانات المتقدمة

الهدف

فهم كيفية تخزين البيانات وتنظيمها للوصول إليها بكفاءة.

المهارة

اختيار هيكل البيانات المناسب وكتابة نماذج Python بسيطة.

السؤال الكبير

لماذا لا نستخدم قائمة واحدة لكل المشكلات؟

Stack – المكدس

المفهوم

هيكل بيانات يعمل بمبدأ LIFO: آخر عنصر يدخل هو أول عنصر يخرج.

مثال حياتي

رص الكتب أو الأطباق: آخر كتاب تضعه في الأعلى هو أول كتاب تستطيع أخذه.

عمليات أساسية

Push للإضافة، Pop للإزالة، Top لمعرفة قمة المكدس.

محاكاة Stack

myStack = []
myStack.append(10)
myStack.append(20)
print(myStack.pop())
print(myStack)
20 [10]
append تضيف في نهاية القائمة، وpop تحذف آخر عنصر؛ لذلك ظهر 20 أولاً لأنه آخر عنصر تمت إضافته.

Queue – الطابور

المفهوم

هيكل بيانات يعمل بمبدأ FIFO: أول عنصر يدخل هو أول عنصر يخرج.

مثال حياتي

طابور المقصف أو الحافلة: من يصل أولاً يحصل على الخدمة أولاً.

عمليات أساسية

Enqueue للإضافة في الخلف، Dequeue للإزالة من الأمام.

محاكاة Queue

queue = []
queue.append("Ali")
queue.append("Omar")
print(queue.pop(0))
print(queue)
Ali ['Omar']
pop(0) يزيل أول عنصر في القائمة، لذلك خرج Ali لأنه دخل الطابور أولاً.

Linked List – القائمة المرتبطة

المفهوم

مجموعة عقد، كل عقدة تحتوي على بيانات ورابط يشير إلى العقدة التالية.

مثال حياتي

عربات القطار: كل عربة تحمل محتوى وترتبط بالعربة التالية.

متى تفيد؟

عند الحاجة إلى إدراج وحذف عناصر بمرونة دون تحريك كل العناصر.

محاكاة Linked List

Trees – الأشجار

المفهوم

هيكل غير خطي يبدأ من Root وتتفرع منه عقد فرعية.

مثال حياتي

شجرة العائلة أو مجلدات الحاسوب: أصل ثم فروع ثم فروع أصغر.

مصطلحات

Root، Parent، Child، Leaf، Subtree.

Graphs – المخططات

المفهوم

مجموعة عقد Nodes وروابط Edges، وقد تكون الروابط موجهة أو غير موجهة.

مثال حياتي

خرائط جوجل: المدن عقد، والطرق بين المدن حواف.

استخدامات

شبكات التواصل، شبكات الحاسوب، الخرائط، علاقات البيانات.

الوحدة الثانية: أدوات التطوير البرمجي وتقنيات الاختبار

الهدف

فهم أدوات بناء البرامج وتنظيمها واختبارها.

المهارة

قراءة كود Python وتحليله وتصحيح الأخطاء.

السؤال الكبير

كيف نضمن أن البرنامج يعمل بشكل صحيح؟

لغات البرمجة عالية المستوى

لغة الآلة

تعليمات ثنائية يصعب على الإنسان قراءتها.

لغة التجميع

أقرب قليلاً للإنسان لكنها مرتبطة ببنية الحاسوب.

مثال حياتي

كما يحتاج الإنسان مترجماً لفهم لغة أجنبية، يحتاج الحاسوب مترجماً أو مفسراً لفهم الكود.

اختر نوع أداة الترجمة.

أدوات تطوير البرمجيات

Code Editor

أداة خفيفة لكتابة الكود.

IDE

بيئة متكاملة للكتابة والتشغيل والتصحيح وإدارة المشروع.

مثال حياتي

محرر الكود كدفتر كتابة، أما IDE فهو ورشة كاملة فيها أدوات متعددة.

Modular Programming – البرمجة التركيبية

الفكرة

تقسيم البرنامج إلى وحدات أو دوال صغيرة يسهل فهمها واختبارها.

مثال حياتي

تنظيم رحلة مدرسية: النقل، التصاريح، التغذية، الإعلام؛ كل جزء له فريق مستقل.

الفائدة

تقليل التكرار، سهولة الصيانة، وضوح الحل.

def average(a, b, c):
    return (a + b + c) / 3

print(average(80, 90, 100))
90.0
الدالة average تستقبل ثلاث قيم، تجمعها، ثم تقسمها على 3.

Libraries – المكتبات البرمجية

المفهوم

مجموعة دوال جاهزة يمكن استدعاؤها عند الحاجة.

مثال حياتي

صندوق الأدوات: لا تصنع المفك كل مرة، بل تستخدم أداة جاهزة.

أمثلة

math، random، datetime، sys.

import math
print(math.sqrt(81))
9.0
استوردنا مكتبة math واستخدمنا الدالة sqrt لحساب الجذر التربيعي.

Runtime Error

Logic Error

البرنامج يعمل لكن النتيجة غير صحيحة.
